# Lost & Found Project

Lost & Found Website adalah aplikasi berbasis web yang digunakan untuk mencatat dan mengelola data barang hilang dan barang ditemukan. Aplikasi ini dibuat untuk memudahkan pengguna dalam menyimpan informasi barang secara terstruktur serta melakukan pengelolaan data dengan cepat dan sederhana. Tujuan pembuatan website ini adalah untuk menerapkan konsep dasar pengembangan web, khususnya penggunaan ReactJS serta pemanfaatan Local Storage sebagai media penyimpanan data tanpa menggunakan database eksternal.

## Fitur Utama

- Menambahkan data barang hilang dan barang ditemukan
- Menampilkan daftar barang dalam bentuk list
- Mengedit data barang
- Menghapus data barang
- Penyimpanan data menggunakan Local Storage browser

## Teknologi yang Digunakan

- ReactJS
- Vite
- JavaScript
- HTML
- CSS
- Local Storage

## Catatan

Proyek ini menggunakan Local Storage, sehingga data hanya tersimpan di browser pengguna dan akan hilang jika cache browser dibersihkan.
